HAVEN AESTHETICS WEBSITE - GO LIVE STEPS

1. Extract this ZIP.
2. Open the extracted folder in VS Code.
3. Confirm these files are visible:
   index.html
   styles.css
   script.js
   assets folder

4. In the VS Code terminal, run:
   git status
   git add .
   git commit -m "Update Haven Aesthetics website assets"
   git push

5. If Git says this is not a repository, run:
   git init
   git branch -M main
   git add .
   git commit -m "Initial Haven Aesthetics website"
   git remote add origin https://github.com/marcoduplessis03-crypto/haven-aesthetics-website.git
   git push -u origin main

6. If GitHub rejects because the remote has files, run:
   git fetch origin
   git push -u origin main --force-with-lease

7. Go to GitHub > repo > Settings > Pages.
   Source: Deploy from a branch
   Branch: main
   Folder: /root
   Save.

8. Wait 1-2 minutes, then open:
   https://marcoduplessis03-crypto.github.io/haven-aesthetics-website/

9. Hard refresh with Ctrl + F5.
